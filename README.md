# libraryCpfValidator
Esta biblioteca destina-se à validação de números de CPF.

## Como instalar:
$  npm install libraryCpfValidator

## Como utilizar:

<p>> const cpfValidator = require('libraryCpfValidator'); <br>
> console.log(cpfValidator('12345678909'));<br>
> // returns 'true'</></p>
